[Perseus Vault](https://github.com/Perseus-Computing-LLC/perseus-vault) is a local-first, single-binary MCP memory engine for AI agents. It combines SQLite + FTS5 keyword search with dense vector (hybrid) search, optional AES-256-GCM encryption, and runs fully offline with no external services. The `langchain-perseus-vault` package exposes it to LangChain as agent tools, a retriever for RAG chains, and a low-level MCP client.

## Installation and setup

<CodeGroup>
```bash pip
pip install langchain-perseus-vault
```

```bash uv
uv add langchain-perseus-vault
```
</CodeGroup>

Perseus Vault runs locally and requires no API key. Point the client at a database file (created on first use):

```python
from langchain_perseus_vault import PerseusVaultClient

client = PerseusVaultClient(db_path="~/.langchain/perseus-vault.db")
```

## Tools

```python
from langchain_perseus_vault import create_perseus_vault_tools

tools = create_perseus_vault_tools(client)
# -> [perseus_vault_remember, perseus_vault_recall]
```

## Retriever

```python
from langchain_perseus_vault import PerseusVaultRetriever

retriever = PerseusVaultRetriever(client=client, k=5)
docs = retriever.invoke("What did we decide about the launch?")
```
