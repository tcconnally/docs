The Headless Fleet API is the programmatic interface to LangSmith Fleet. Use it to build custom UIs, integrate Fleet into existing applications, or manage agents from code. For example, you can build:

- Agent products embedded in your existing application, with your own branding and authentication.
- Internal tools and admin consoles for your engineers and operators.
- Agent provisioning pipelines driven from infrastructure-as-code or CI.

<Card title="Quickstart" icon="rocket" href="/langsmith/fleet/headless-fleet-quickstart">

<Info>
  [Fleet](/langsmith/fleet) is the no-code product for building and running agents from a hosted UI. The Headless Fleet API exposes the same underlying platform as a set of REST endpoints. Resources created through Fleet are available through the API, and resources created through the API appear in the Fleet UI.
</Info>

Browse the full API reference in the **Headless Fleet API** section in the sidebar, or see the endpoint groups below:

- [Agents](/langsmith/fleet/headless-fleet-api/agents): Configured graphs with their own files, configuration, and metadata
- [Threads](/langsmith/fleet/headless-fleet-api/threads): Accumulated state of a conversation across one or more runs
- [Skills](/langsmith/fleet/headless-fleet-api/skills): Reusable, versioned bundles of files that agents can load at runtime
- [Integrations](/langsmith/fleet/headless-fleet-api/integrations): Third-party services available to your workspace, with one or more supported auth methods
- [Auth providers](/langsmith/fleet/headless-fleet-api/auth-providers): OAuth providers that authorize connections on behalf of your users
- [Auth sessions](/langsmith/fleet/headless-fleet-api/auth-sessions): In-flight OAuth flows that connect an end user to a provider
- [Auth tokens](/langsmith/fleet/headless-fleet-api/auth-tokens): Connection tokens representing a user's authorized access to a third-party service
- [Triggers](/langsmith/fleet/headless-fleet-api/triggers): Events that fire an agent on a schedule, an inbound message, or a webhook
- [Trigger templates](/langsmith/fleet/headless-fleet-api/trigger-templates): Kinds of triggers available to your workspace, such as cron, webhook, and Slack bot
